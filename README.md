# java-cli-bazel-h2-simple

## Description
A java bazel build, that connects to h2
database.

H2 is a database written in `java`
and is best known for a `dev` only
use.

## Tech stack
- java
- bazel
- h2 driver

## Docker stack
- l.gcr.io/google/bazel:latest

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
